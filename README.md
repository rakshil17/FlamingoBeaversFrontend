# FlamingoBeaversFrontend
Flamingo Beavers Frontend

Basically npm install then npm dev run then running the app.jsx to start the frontend to connect to backend you will also need the backend server running simultaneously 


## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the application
```bash
npm run dev
```

### 3. (Optional) Run the deployed version (backend not deployed yet will be deployed after 23rd March)
Frontend only: https://flamingo-beavers-frontend-deploy.vercel.app/
